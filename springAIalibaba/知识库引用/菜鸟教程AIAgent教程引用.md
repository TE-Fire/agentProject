# 菜鸟教程 AI Agent 教程引用库

## 一、引用库用途

本引用库用于收集和整理菜鸟教程（runoob.com）上关于 AI Agent 开发的相关教程链接，方便学习和查阅。

**主要用途**：
1. **学习参考**：作为 Spring AI Alibaba 学习过程中的外部知识来源
2. **案例对照**：结合实际代码案例进行对照学习
3. **知识拓展**：了解 AI Agent 领域的基础概念和实践技巧
4. **问题解答**：遇到概念性问题时，可快速查阅相关教程

## 三、引用规范

```
【知识点名称】：参考菜鸟教程《【教程标题】》，链接：【URL】
```
