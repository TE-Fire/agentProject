流式输出技术对比演示

对比 SSE、WebSocket、Fetch+ReadableStream 三种流式通信技术

SSE

Server-Sent Events

特点：服务器主动推送，单向通信，适合 AI 打字机效果
WebSocket

WebSocket 双向通信

特点：全双工通信，支持双向实时交互
Fetch

Fetch + ReadableStream

特点：现代浏览器原生支持，灵活控制流解析

三种流式技术对比

特性 SSE WebSocket Fetch + Stream
通信方向 单向（服务端推送） 双向（全双工） 单向（客户端拉取）
浏览器兼容性 IE 不支持 良好 现代浏览器
实现复杂度 低 中 中
自动重连 支持 需手动处理 每次请求新建
适用场景 AI 打字机、进度通知 实时聊天、协同编辑 自定义流解析
数据格式 text/event-stream 文本/二进制 任意流式数据
性能 高 高 高
